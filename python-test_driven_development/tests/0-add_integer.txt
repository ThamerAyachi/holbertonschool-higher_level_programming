Using ``add_integer``
--------------------

Parameter "b" is default to int 98
Make sur the parameter are type int
Return the sum of the parameter

First import ``add_integer``

    >>> add_integer = __import__("0-add_integer").add_integer

Now use it:

Basic usage:
    >>> add_integer(1, 2)
    3

Negative input:
    >>> add_integer(3, -1)
    2

One input:
    >>> add_integer(2)
    100

Float and negative input:
    >>> add_integer(1.2, -1)
    0

One float input:
    >>> add_integer(2.1)
    100
